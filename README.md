# Mapping-Discourse-Relations

Code used to obtain the results presented in the paper "Mapping Explicit and Implicit Discourse Relations between the RST-DT and the PDTB 3.0" published in the Proceedings of the 14th International Conference on Recent Advances in Natural Language Processing (RANLP 2023).

> https://aclanthology.org/2023.ranlp-1.39/

Authors: Nelson Filipe Costa, Nadia Sheikh and Leila Kosseim.
